**UndertaleModLoader (English desc):**
*UndertaleModLoader was created specially for Undertale Modding Community.*

**v0.1.0**
What feautures our engine provides?
Fully custom programming language interpreter(sort of)
DLL functions support
You can setup our mod on any version of undertale(gms2.0.6+)
You can replace any texture you want in the game
You can make custom npc with custom texture and dialogue
You can add custom sprite(useless now)

**Contributing**
If you would do changes for the Engine, you need to overwrite files in UTMT directory.
You can add new objects, but write it to the ApplyPatch script too.
